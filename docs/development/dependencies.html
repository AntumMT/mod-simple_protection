Dev dependencies

Simple Protection requires to following mods to work correctly.

Dependencies

default Base mod for many games.
intllib (optional) Internationalization library.
areas (optional) Advanced area protection.


Dependencies

default
Base mod for many games. Included in minetest_game.

Internationalization library.

Provides internationalization support for mods.

Advanced area protection.
