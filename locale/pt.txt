#### AREA COMMAND ####

# Command description
Manages all of your areas. = Gerencia todas as suas areas.

# /area
Available area commands: = Comandos disponiveis:
Information about this area: /area show = Dados sobre essa area: /area show
(Un)share one area: /area (un)share <name> = (Des)compartilha uma area: /area (un)share <nome>
(Un)share all areas: /area (un)shareall <name> = (Des)compartilha todas areas: /area (un)shareall <nome>
Unclaim this area: /area unclaim = Desfazer-se dessa area: /area unclaim
